1. **Create `package.json`**
    ```npm init```
    
2. **Install cypress**
    ```npm install cypress --save-dev```
    
3. **Cypress latest version**
    ```npm install --save-dev cypress@latest```
    
4. **Open cypress**
    ```./node_modules/.bin/cypress open```
